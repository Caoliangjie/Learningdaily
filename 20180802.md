## 天天向上的力量
```
dayfactor=0.01
dayup=1.0
for i in range(365):
    if i %7 in [6,0]:# xiuxiri
        dayup=dayup*(1-dayfactor)
    else:
        dayup=dayup*(1+dayfactor)
print("liliang:{:.2f}".format(dayup))

```
用一个代码来解释这个天天向上的问题，这里是考虑了双休日和工作日，之后记录就变成了结果4.63（每天1%努力会增加这些效率）

 比如，{0：e}。format（3.14）输出为3.140000e+00（科学计数法），更改类型后可以控制数据输出结果。
